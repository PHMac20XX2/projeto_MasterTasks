# MasterTasks - Gerenciador de Tarefas

O **MasterTasks** é uma aplicação focada em produtividade, com foco em permitir que usuarios tenham como gerenciar as tarefas do dia a dia de forma simples e agíl.

---

## Tecnologias Utilizadas

Nesse projeto, usei as seguintes tecnologias:

- **HTML & CSS3:** Estrutura junto a estilização personalizada.
- **JavaScript(JS):** Lógica de validação e manipulação.
- **Git & Github:** Controle de versão e organização de branchs.
